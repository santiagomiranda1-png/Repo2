# ClaseLLm
# Requerimientos chatbot de banco
# Requerimientos funcionales
1. Dar las respuestas correctas sin aluciar
2. soportar grandes cantidades de solicitudes a la vez
3. responder acertadamente a preguntas relacionadas al historial he informacion de la cuenta
4. chat cifrado para la mayor seguridad y privacidad posible
5. seguridad y cifrado del código

# No funcionales
1. Dar respuestas en menos de 10 segundos
2. Estar disponible la mayor cantidad de tiempo posible
3. Memoria que soporte gran cantidad de informacion en un chat
4. opcion de reportar en caso de error.
5. interfaz personalizable
